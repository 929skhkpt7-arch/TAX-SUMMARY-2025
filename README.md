# 2025 ETF & Fund Tax Treatment

A searchable table of community-reported 2025 distribution tax-treatment data.

The table contains 74 records covering 54 unique ticker symbols. It supports ticker search, asset-category filtering, portrait PDF printing, and filtered CSV/Excel exports.

## Open the table

Open `index.html` in a browser.

## GitHub Pages

Upload `index.html` and this `README.md` directly to the root of the repository's `main` branch. Then enable GitHub Pages under **Settings → Pages** and select:

- Branch: `main`
- Folder: `/ (root)`

> Educational information only. Verify every entry against your final corrected tax documents before filing.
